    Skype

    Definition : the original concept for the name was Sky-Peer-to-Peer, which morphed into Skyper, then Skype
    Category : Computing » Software & Applications
    Country/Region : Worldwide
    Popularity :

    What does Skype mean?

    Skype is a software application that allows users to make voice and video calls and chats over the Internet. One of the initial names for the project was “Sky peer-to-peer”, which was then abbreviated to “Skyper”. However, some of the domain names associated with “Skyper” were already taken. Dropping the final ‘r’ left the current title “Skype“, for which domain names were available.
