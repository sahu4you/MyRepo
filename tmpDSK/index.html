    tmpDSK

    Definition : Temporary Disk
    Category : Computing » Data Storage
    Country/Region : Worldwide
    Popularity :

    What does tmpDSK mean?

    tmpDSK means Temporary Disk in Cpanel. /usr/tmpDSK is your temporary system folder that stores temp files.
