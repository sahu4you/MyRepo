    Definition : PHP Data Objects
    Category : Computing » Databases
    Country/Region : Worldwide
    Popularity :

    What does PDO mean?

    PHP Data Objects (PDO) is a lightweight system to access data from databases with PHP. PDO provides a data-access abstraction layer, which means that, regardless of which database you’re using, you use the same functions to issue queries and fetch data. Thus PDO allows you to access a variety of different database and provides an easy way to query, retrieve results. PDO requires a minimum php version of 5.1
